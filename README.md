# Actividad Semana 13 Formulario de Contacto

## Objetivo

Desarrollar un formulario de contacto utilizando HTML, CSS y JavaScript, aplicando validaciones y manipulación del DOM.

## Archivos del proyecto

- index.html
- style.css
- script.js

## Instalación

1. Descargar o clonar el repositorio desde GitHub.
2. Extraer la carpeta del proyecto.
3. Copiar la carpeta `actividad_3_1_frontend` dentro de `C:\xampp\htdocs\`.
4. Iniciar Apache desde XAMPP.

## Ejecución

Abrir el navegador e ingresar:

```
http://localhost/actividad_3_1_frontend/
```

## Funcionalidades

- Validación de campos obligatorios.
- Validación del formato del correo electrónico.
- Mensajes de error junto a cada campo.
- Mensaje de confirmación al enviar correctamente.
- Limpieza automática del formulario.
